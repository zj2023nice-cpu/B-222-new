# 校园心理健康小程序 (Campus Mental Health)

## 项目简介

本项目是一个基于微信小程序云开发构建的校园心理健康服务平台。旨在为学生提供便捷的心理咨询预约、心理测评、心情记录以及心理健康知识科普等服务，同时为心理咨询师和管理员提供相应的管理功能。

## 核心功能

### 1. 角色体系

系统包含三种主要角色，拥有不同的权限和功能界面：

- **学生端**：预约咨询、浏览文章、参与测评、心情打卡、AI 智能问答。
- **咨询师端**：管理预约、发布/管理测评、个人资料管理。
- **管理员端**：用户管理、内容管理（文章、测评）、资讯配置。

### 2. 主要模块

- **首页**：推荐文章、快捷入口、今日心情。
- **预约系统**：支持查看咨询师列表、预约时间段选择、预约状态管理。
- **心理测评**：提供多种心理量表测评，自动评分并生成记录。
- **知识科普**：心理健康文章的浏览与收藏。
- **情绪日记**：记录每日心情，生成情绪记录。
- **AI 咨询助手**：接入 DeepSeek 大模型，提供即时的心理健康咨询与对话支持。

## 技术栈

- **前端框架**：微信小程序原生开发 (WXML, WXSS, JS)
- **UI 组件库**：[TDesign Miniprogram](https://tdesign.tencent.com/miniprogram)
- **后端服务**：微信云开发 (WeChat CloudBase)
  - **云函数 (Cloud Functions)**：处理业务逻辑 (Node.js)
  - **云数据库 (Cloud Database)**：NoSQL 数据存储
- **AI 能力**：云服务端集成 DeepSeek API 实现智能对话

## 云开发配置

本项目基于微信小程序云开发构建，在使用前需完成以下配置步骤：

### 1. 开通云服务与获取环境 ID

1. 在微信开发者工具工具栏，点击 **[云开发]** 图标。
2. 按照指引开通云服务，并创建一个云开发环境。
3. 进入云开发控制台，在 **[设置] -> [环境设置]** 中复制 **环境 ID**。

### 2. 配置小程序环境 ID

1. 打开项目根目录下的 `app.js`。
2. 将 `wx.cloud.init` 中的 `env` 字段值替换为你自己的 **环境 ID**。
   ```javascript
   wx.cloud.init({
     env: "your-env-id",
     traceUser: true,
   });
   ```

### 3. 云函数部署

1. 在开发者工具目录树中，右键点击 `cloudfunctions` 文件夹。
2. 选择 **“当前环境内安装并上传所有云函数”**，确保所有服务模块部署成功。

### 4. 数据库配置

进入云开发控制台的 **[数据库]** 模块，手动新建以下集合：

1. `users`: 用户基础信息
2. `admins`: 管理员账号
3. `consultants`: 咨询师列表
4. `articles`: 心理科普文章
5. `appointments`: 预约申请记录
6. `assessments`: 测评问卷题目
7. `assessment_records`: 咨询师评估学生的咨询记录
8. `student_test_records`: 学生自主完成的心理测评记录
9. `mood_diaries`: 用户心情日记
10. `carousel`: 首页轮播图配置
11. `user_collections`: 用户收藏的文章列表

### 5. 启用 AI 模型

本项目 AI 助手使用云开发内置的 DeepSeek 模型，需手动开启：

1. 进入**云开发控制台**，点击左侧菜单栏的 **[AI]**。
2. 在顶部标签页选择 **[生文模型]**。
3. 找到 **deepseek** 分组，点击对应的 **[启用]** 按钮。
4. 状态显示为 **“已启用”** 后，返回开发工具，清除小程序**全部缓存**，并**重新编译**小程序，AI 咨询功能即可正常使用。

## 快速开始

1. **环境准备**：
   - 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 并登录。
   - 完成上述 **[云开发配置]** 步骤。

2. **依赖安装**：
   - 在微信开发工具导入项目，并在终端的项目目录下运行 `npm install` 安装相关依赖。
   - 在开发者工具中点击 **工具 -> 构建 npm** ，等待构建完成。

3. **手机预览：**
   - 点击开发者工具中的 **预览** 按钮，等待代码打包上传，即可微信扫码或自动预览。

## 注意事项

- 项目使用了 **TDesign** 组件库，请确保已安装依赖，并成功执行 **npm 构建**。
- 云函数已变更，需点击 `cloudfunctions` 文件夹，选择**同步云函数列表**。
- 已补充需要新建的集合，请检查数据库中的集合列表是否匹配。
